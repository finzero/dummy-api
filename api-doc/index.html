Authentication

LOGIN

POST
/auth/login

Request body :
{
    "userId": "userId",
    "password": "password"
}

Response :
{
    "errorCode": "00",
    "errorMessage": "success login",
    "data": {
        userId: 'userid',
        token: 'token',
        refreshToken: 'refreshToken',
        expires: 3600
    }
}

BODY PARAMETERS

Field Type Is Required Description
username String true -
password String true -
